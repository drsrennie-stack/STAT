The CODE cycle, in one screen

Lesson 1.5 walks you through it

Every week of this cohort runs the same four-phase loop. You learn it this week and run it in your own classes the rest of your career.

01

Clarify

Pin down what is actually wrong before reaching for a fix. Most students skip this and pay for it later.

02

Organize

Rebuild the structure of what you know so retrieval is fast and confident under exam pressure.

03

Drill

Targeted retrieval practice on the exact gaps the first two phases surfaced. No wasted reps.

04

Evaluate

Honest weekly accounting. What worked, what did not, what changes next week.
